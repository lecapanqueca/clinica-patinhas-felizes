# 🐾 Patinhas Felizes

Site institucional de uma clínica veterinária fictícia, desenvolvido com **HTML e CSS**, focado em layout moderno, organização e responsividade.

## 📚 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio da trilha de HTML da plataforma DIO.

O objetivo foi criar um site completo aplicando os conceitos aprendidos no módulo de HTML básico, incluindo:

- Estruturação de páginas com HTML
- Uso de formulários
- Tabelas
- Inserção de mídias
- Organização de conteúdo em múltiplas páginas
- Navegação entre páginas

## ✨ Funcionalidades

- Página inicial com apresentação da clínica  
- Página "Sobre a Clínica"  
- Tabela de horários estilizada  
- Página de contato com formulário  
- Layout consistente entre páginas  
- Uso de ícones e fontes personalizadas  
- Estrutura organizada e reutilizável  

---

## 🎨 Layout

O projeto foi desenvolvido com foco em:

- Cores suaves e amigáveis (tons de rosa e neutros)  
- Tipografia moderna  
- Elementos com bordas arredondadas  
- Sombras suaves para destaque  
- Experiência visual agradável  

---
